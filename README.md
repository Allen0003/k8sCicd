This project demonstrates a robust, automated CI/CD pipeline for a decoupled full-stack application. It focuses on containerization, automated testing, and zero-touch deployment using GitHub Actions and Render.

Project Objectives
1. Automated Testing: Ensure code reliability using JUnit and JaCoCo coverage reports.
2. Containerization: Standardize environments for both Spring Boot (Backend) and Angular (Frontend + Nginx).
3. CI (Continuous Integration): Automated builds and image pushes to GitHub Container Registry (GHCR).
4. CD (Continuous Deployment): Automated triggering of Render Deploy Hooks for instant updates.
